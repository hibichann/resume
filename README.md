* **降低内存占用：** 在项目测试阶段，发现某些页签在打开时会导致内存占用急剧增加。经过分析，我发现问题出在某些下拉选择树列表的DOM数量过大。为解决这个问题，我们将其改为懒加载，从而成功地将内存使用率降至正常水平。
